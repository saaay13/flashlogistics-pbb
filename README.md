# FlashLogistics DSS

Sistema de apoyo a la toma de decisiones (DSS) para optimización de rutas de distribución y trazabilidad en tiempo real de flotas.

## Épicas

- **Gestión de Datos** - Administración de pedidos, conductores y vehículos
- **Motor de Rutas** - Algoritmo de optimización y cálculo de costos
- **Monitoreo GPS** - Seguimiento en tiempo real y geolocalización
- **Dashboard y Reportes** - Indicadores clave y toma de decisiones
- **Notificaciones** - Alertas automáticas a clientes y despachadores
- **App Conductores** - Aplicación móvil para el equipo de reparto
